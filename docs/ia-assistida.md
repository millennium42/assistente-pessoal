# IA Assistida no Desenvolvimento

O projeto declara uso de IA assistida como ferramenta de desenvolvimento.

## O que isso significa

- IA ajuda a revisar arquitetura, gerar codigo, propor testes e escrever documentacao.
- O codigo precisa passar por revisao humana.
- Testes automatizados continuam obrigatorios.
- Seguranca e privacidade nao sao delegadas a IA.

## Onde fica visivel

- Neste documento.
- README.
- Rodape discreto do dashboard desktop.

## Limites

Nao se deve afirmar que a IA garante:

- conformidade juridica;
- ausencia de bugs;
- seguranca perfeita;
- qualidade de producao sem revisao humana.

## Pratica esperada

- Commits pequenos e revisaveis.
- Testes para regressao.
- Secret scanning.
- Auditoria de dados sensiveis.
- Documentacao das decisoes tecnicas.
